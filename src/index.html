Troubleshooting


False AdBlock Detection

FalixNodes Desktop does not contain an adblocker, therefore if FalixNodes detects an adblock, it is due to another factor.

To begin, we'll check for a few things that may be blocking ads on your system affecting FalixNodes Desktop.

Check Your Anti-Virus

Some anti-virus software these days now come with a system-wide adblocker, which will affect FalixNodes Desktop. If you have an anti-virus, please check it and find the AdBlock settings if any. If the anti-virus software does have an AdBlock system, either disable the option, or if optional whitelist FalixNodes Desktop to allow ads.

If you do not have anti-virus or it's not to blame, move on to the next step.

Check Your System Hosts File

As a last resort, we noticed customers may of modified their hosts file to block ads system-wide. Modifying the hosts file is done for common reasons for like blocking ads in Spotify and other applications, doing so will also affect FalixNodes Dekstop.

Hosts file is used to store mappings of hostnames to IP addresses. This file is consulted first when resolving hostnames, so it can be used to override the results from DNS lookups.

By clicking "Reset Hosts File" below, you agree to FalixNodes Desktop resetting your system's hosts file. As a reminder, FalixNodes Desktop does not come with wararnty and is not liable for any damage to your system. The function has been tested on Windows 10 and Linux.

Running FalixNodes Desktop as admin is required on Windows for the reset hosts file button to fully work.

System is Slow

There are several reasons why your system may be performing slowly. You may have too many applications open, insufficient resources to tackle more processes, and other problems. This troubleshooting may help you speed up your system utilizing simple actions.

Optimize FalixNodes Desktop

Before we get started on boosting your computer's performance, let's have a look at FalixNodes Desktop. Let's make a few adjustments to FalixNodes Desktop's settings to help it consume fewer resources. Click "Adjust Settings" below to:

  • Toggle Windows Blur off
  • Toggle Animations off
  • Don't load Servers Panel by default (Will unload Servers Panel)
  • Disable tabs: Help Center, Support, and Account (in Settings)
  • Toggle all Dashboard content off
    •

    Most of the steps below are intended to be done on Windows 10 and Windows 11. FalixNodes Desktop does not officially support Windows 7 or Windows 8.1, let alone any other older versions of Microsoft Windows.

    Speeding up Windows

    Taking Out The Trash

    There are a few places on Windows that you should delete, starting with the most obvious one being your Recycle Bin.


    Recycle Bin

    Before you go emptying the Recycle Bin, let's learn more about it first and see if you really need to empty it in the first place.

    The Recycle Bin is a folder or directory in Windows where deleted files are temporarily stored. Unless they are too large, deleted files are not permanently removed from the hard disk but instead placed to the Recycle Bin. The Recycle Bin files can be restored to their original place.

    Obviously, empty the Recycle Bin, but be sure the files in there are no longer needed.


    Temporary Files

    Temp files are files that your operating system only required to exist for just a short period of time while in use but are now basically taking up space. The majority of temporary files are stored in the Windows Temp folder, which differs per machine.

    To find and delete all temp files, press + R and run %temp%. Running this will open up Windows File Explorer and reveal where the temp folder is stored on your machine.

    Select all files and delete. If a dialog shows up saying "File in Use", just select "Do this for all current items" and click "Skip".

    Uninstall Unused Applications

    You probably have some, or a lot, of unused applications you either didn't ever use or simply don't use anymore. Some Windows PCs even come with a bunch of software pre-installed, also known as bloatware.

    Open your System Settings > Apps > Then uninstall apps you no longer need.

    Otionally, you can also remove built-in programs that come with Windows like Internet Explorer, Paint, and more. You can do this by going to System Settings > Apps > Optional Features > and uninstall Internet Explorer and/or more.

    Settings Adjustment

    Disable Startup Programs

    Some applications, such as Discord, Spotify, and Steam, will automatically launch when you log onto Windows after powering on your computer or logging out and back in. You can configure this by either changing the option in the application's preferences or changing it in your Task Manager.

    To access your Task Manager, right-click in an empty space on your Taskbar, then click on "Task Manager". If you don't have empty space left on the taskbar, addtionally you can also right-click on the Start button and access Task Manager from there as well.

    In the Task Manager, toggle the option on or off on applications you don't want starting up by default in the "Startup" tab.

    Adjusting Performance Settings

    Many visual effects, such as animations, blur, and shadow effects are included in Windows 10 and later. These are visually appealing, but they can consume additional system resources and slow down your computer. This is especially the case if you have a PC with very little memory (RAM).

    Luckily, these visual effects can be disabled to free up additional system resources, helping your PC run a little faster than it did before.

    Open your File Explorer > right-click on "This PC" > click "Properties". A new window should appear that displays your system's information, click on "Advanced system settings" located on the left side.

    In the Advanced tab, click on "Settings..." under Performance. Then, in the Visual Effects tab, disable the following:

    • Animate contorls and elements inside windows
    • Animate windows when minimizing and maximizing
    • Animations in the taskbar
    • Enable Peek
    • Fade or slide menus into view
    • Fade or slide ToolTips into view
    • Save taskbar thumbnail previews
    • Show shadows under mouse pointer
    • Show shadows under windows
    • Show translucent selection of rectangle
    • Slide open combo boxes
    • Smooth-scroll list boxes
    • Use drop shadows for icons labels on the desktop

    Then, make sure the following remains enabled:

    • Show thumbnails instead of icons
    • Show window contents while dragging
    • Smooth edges of screen fonts

    Adjust Appearance Settings

    As mentioned before, blur is one of the visual effects that comes with Windows, since the release of Microsoft Windows 7. This can also take up performance on your CPU, you can disable it in the System Settings app.

    Open System Settings > Personalization > Colors > and disable Transparency effects. Also make sure the blur effect is disabled too in the FalixNodes Desktop appearance settings, even if Transparency effects is disabled on Windows's side, the blur effect in FalixNodes Desktop is forced if left enabled.
